What is Angular?

AngularJS is a library that you can use to greatly enhance the ability of HTML and it's functionality. HTML on its own -- even when paired with JavaScript -- is often not powerful enough to fulfill the needs of developers today. AngularJS introduces functionality such as data-binding, views, and so much more.
Angular on its own is very powerful. However, there are many different plugins that you can choose to incorporate with Angular. When you pair this with Bootstrap or any other common framework/library available to developers, Angular makes your web development arsenal a force to be reckoned with.

Routing and Views

AngularJS uses something very similar to ASP.NET's MVC framework. It uses different "views" to render what is on the screen dynamically. In our case, index.html acts as a sort of layout for every page. This includes the footer, the navbar at the top of this page, and other similar items. All of the scripts and CSS styling documents needed to make the page work are included in the index.html as well.

If you look here, you can see that our current views are about.html and main.html. This means that the site knows about two views that can be displayed to the user. Let's create another view.

Create a new HTML file in the views directory titled tutorial.html. You'll also need to create a new file on the scripts/controllers file named tutorial.js and declare a controller named "TutorialCtrl" in that file.

Once you've done that, we need to tell our Angular application that you've created a new view and controller for that view. This is what is called routing. If you try to navigate to a route that you've defined, Angular will take that and render the view in the section of your index.html that you specify for views. Add this snippet of code to your app.js file below your angular.module declaration:

.when('/tutorial', {
          templateUrl: 'views/tutorial.html',
          controller: 'TutorialCtrl',
          controllerAs: 'tutorial'
          })

Once this is done, create a list item in your navbar in your index.html with an ng-href of "#/tutorial". Once you've done this, your Angular app will know about the route you've defined for the tutorials view and will render it whenever you click your link in the navbar for it! Easy, right?
